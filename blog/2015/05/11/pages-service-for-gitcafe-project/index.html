Pages Service for GitCafe Project

May 11, 2015 #dev

本文简单介绍如何在GitCafe的项目中使用Pages服务

参考网址：WIKI for GitCafe Pages

1.在GitCafe上新建项目，假设名为resourcerepository

2.克隆项目到本地，并进入该项目目录

git clone git@gitcafe.com:hujiawei/resourcerepository.git

3.新建分支gitcafe-pages，然后提交该分支

git checkout -b gitcafe-pages
git push -u origin gitcafe-pages

4.添加一个测试文件，例如index.html

echo "Hello, hujiawei" > index.html

5.提交测试文件到GitCafe

git add index.html
git commit -m "test"
git push

6.通过访问username.gitcafe.io/projectname/，例如http://hujiawei.gitcafe.io/resourcerepository/即可看到页面显示Hello, hujiawei
